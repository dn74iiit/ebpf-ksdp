# eBPF-KSDP: Profiling Kernel Scheduling Delays for Network-Heavy Workloads

## Project Overview
This project leverages **eBPF** (Extended Berkeley Packet Filter) to profile and analyze kernel scheduling latencies in environments with high network throughput. By hooking into kernel tracepoints, we aim to quantify how network-intensive tasks impact task scheduling and overall system responsiveness.
